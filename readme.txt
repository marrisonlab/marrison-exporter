Plugin per esportare gli ordini di WooCommerce in formato CSV con selezione di date e colonne personalizzate.

== Description ==

Marrison Exporter è un plugin WordPress che permette di esportare facilmente gli ordini di WooCommerce in formato CSV.

Caratteristiche principali:
* Selezione di un range di date per filtrare gli ordini da esportare
* Scelta delle colonne specifiche da includere nell'esportazione
* Interfaccia semplice e intuitiva integrata nel menu di WooCommerce
* Supporto completo per caratteri UTF-8
* Esportazione sicura con verifica dei permessi

== Installation ==

1. Scarica il file del plugin
2. Carica la cartella `marrison-exporter` nella directory `/wp-content/plugins/` del tuo sito WordPress
3. Attiva il plugin dalla pagina "Plugin" in WordPress
4. Assicurati che WooCommerce sia installato e attivato

== Usage ==

1. Vai nel menu **WooCommerce** → **Marrison Exporter**
2. Seleziona un range di date (opzionale)
3. Scegli le colonne che vuoi esportare
4. Clicca su "Esporta Ordini in CSV"
5. Il file CSV verrà scaricato automaticamente

== Columns Available ==

Il plugin permette di esportare le seguenti colonne:
* ID Ordine
* Numero Ordine
* Stato Ordine
* Data Ordine
* Nome Cliente
* Email Cliente
* Telefono Cliente
* Indirizzo Fatturazione
* Indirizzo Spedizione
* Metodo Pagamento
* Metodo Spedizione
* Totale Ordine
* Subtotale
* Tasse
* Spedizione
* Sconto
* Prodotti
* Quantità
* Prezzo Prodotto
* Note Ordine
